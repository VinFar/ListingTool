#Konfigurationsdatei für die Eingabemaske
#Kommentare können mit '#' eingefügt werden. Diese Zeilen werden ignoriert
#Mit '[Kategorie]' (eckige Klammern) können neue Kategorien angelegt werden.
#Diese erzeugen in der Maske einen neuen Tab und jeweils eine neue Datei mit dem jeweiligen Namen
#Darunter können in jeder neuen Zeile eine neue Eigenschaft der Kategorie eingefügt werden.
#Hierbei ist wichtig dass hinter jeden Eigenschaft ein '=' steht.
#Es können beliebig viele Eigenschaften zu jeder Kategorie hinterlegt werden
#Jede Eigenschaft erzeugt eine neue Spalte in der Exel Tabelle
#WICHTIG: Die Excel Datei wird mit ';' Spaltengetrennt.
#Es ist eventuell nötig in den Excel Optionen die Spaltentrenner von ',' auf ';' umzustellen.

#Setup: Python 3.9 installieren
#Rechner Neu starten
#In den Ordner navigieren wo die Dateien liegen
#Oben in die Navigationsleiste klicken und cmd eingeben, dann Enter
#pip install -r requirements.txt
#warten und dann über python ListingTool.py das Skript starte
